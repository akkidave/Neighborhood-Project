Basic Description
This project is using Google Maps API to display the Vadodara, Gujarat area.<br/>
The markers display many of the monuments and main attractions of the Vadodara, Gujarat area. <br/>
<br/>
<strong>Functionality of the page includes:</strong> <br/>
    An interactive navigation bar to filter map and navigation bar results using knockout.js, <br/>
    Display of the wikipedia article of Vadodara, Gujarat using Wikipedia API, <br/>
    Options for filtering the locations based on it's historical presence, <br/>
    Responsive design giving the ability to be used and viewed on all web modern browsing devices

<strong>How to operate the APP:</strong> <br/>
1.Download the project zip file and unzip at some location in your device.
2.Open index.html file in a web browser of your choice (Chrome is more preferred).
3.The application contains a list of places in the city of Vadodara and also those places are shown using markers on the map.
4.To know more about the locations click on the marker or name of the location in the list.
5.User can also filter the locations on the basis of it's historical presence using butoons provided above the list
6.On the top-right corner, a logo of Wikipedia is visible which can clicked to get a link to the Wikipedia article about the city.
7.The application is all device friendly (responsive).
